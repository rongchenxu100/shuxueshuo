# shuxueshuo

中考数学可视化题库网站。

这个项目希望把各地中考真题，以及一模、二模中的典型题，制作成交互式网页，让学生不只是“看答案”，而是能够亲手操作图形，理解题目的变化过程和解题逻辑。

项目的重点不只是展示题目，而是把数学中的动态过程真正做出来。尤其是在几何题中，学生常常难以理解动点、旋转、折叠、轨迹、相似变化等过程。我们希望通过网页交互，把这些原本抽象、静态、难以想象的过程，变成可以观察、可以操作、可以反复体验的学习内容。

## 项目定位

这是一个网站，而不只是若干题目的静态集合。

网站主要由以下几部分组成：

- 首页：介绍项目理念、学习价值和整体入口
- 导航页：按地区、年份、题型、专题等方式组织内容
- 题目页面：每道题一个网页，形成逐步扩展的可视化题库

## 核心理念

我们希望让学生从“被动看题解”转向“主动操作和理解”。

每道题的网页都会尽量围绕以下几个方向展开：

- 图形可操作：支持拖拽、动点、旋转、折叠、缩放等交互
- 过程可观察：让条件变化、图形关系和关键结论动态呈现
- 步骤可同步：图形演示与解题思路同步对应
- 理解可重复：学生可以自己反复操作，重新理解关键过程

## 解决的问题

传统的题目讲解往往以静态图片和文字为主，但很多数学题，尤其是几何题，本质上是动态过程问题。学生常见的困难包括：

- 看不懂图形是怎么变化的
- 不清楚辅助线、旋转、折叠为什么这样构造
- 知道答案但不知道思路是如何一步步形成的
- 听懂了一次，但缺少可以自己重复验证的方式

这个项目希望通过交互式可视化，把这些理解门槛降下来，帮助学生更直观地建立题感、过程感和解题思维。

## 内容形式

题库内容将以“每道题一个网页”的方式组织。

每个题目页面可以包含但不限于以下内容：

- 题目原文与已知条件
- 可交互图形区域
- 动态步骤讲解
- 关键结论高亮
- 多种解法或不同观察角度
- 适合复习的结论总结

这种组织方式既方便单题打磨，也方便后续逐步沉淀成体系化题库。

## 适用人群

- 学生：通过亲手操作图形，更直观地理解题目
- 老师：作为课堂讲解、演示和备课素材
- 家长：在辅导过程中更容易和学生一起观察题目变化

## 题目来源

项目内容主要面向：

- 各地中考真题
- 一模、二模中的典型题
- 适合可视化表达的高价值题目

后续会优先选择那些“静态讲解不够直观、但动态展示效果明显”的题目进行制作。

## 项目目标

我们希望逐步搭建一个真正有学习价值的中考数学可视化题库网站：

- 让抽象过程变得直观
- 让解题步骤和图形关系建立同步理解
- 让学生可以通过操作而不是死记来掌握题目
- 让单题积累最终沉淀成完整题库

## 后续规划

- 完善首页与导航页的信息架构
- 建立统一的题目页面结构
- 优先制作适合动态展示的几何题
- 逐步扩展到更多地区、年份和专题
- 沉淀一套适合数学可视化表达的页面与交互规范
