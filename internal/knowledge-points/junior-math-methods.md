# 初中数学解题方法与题型标签

本文件用于 `geometry-lesson` 与 `quadratic-lesson` 的参考选择。先用 pattern 判断题目大类，再用 method 判断具体解法。这里的 method 面向初中学生讲解，优先使用构造、全等、相似、特殊三角形、折线最短等可视化方法。

## 题型标签 (Pattern)

| pattern ID | 名称 | 典型特征 |
|---|---|---|
| `path-minimum` | 路径最值 | 求折线和、含权重距离和、动点路径最短 |
| `distance-difference-maximum` | 距离差最大值 | 形如 `PA-PM`、`PB-PM` 的最大值 |
| `coefficient-constraint` | 系数约束 | 由点在抛物线、几何条件、线段关系求系数 |
| `moving-point-translation-area` | 动点平移面积 | 图形平移后求重叠面积、面积范围 |
| `moving-point-folding-area` | 动点折叠面积 | 折叠图形与原图重叠面积、阶段范围 |
| `moving-point-rotation-area` | 动点旋转面积 | 旋转图形与原图重叠面积、阶段范围 |
| `parameter-range-inequality` | 参数范围不等式 | 由函数值大小、恒成立不等式求参数范围 |
| `special-triangle-existence` | 特殊三角形存在性 | 判断或构造等腰、直角、等腰直角三角形 |
| `segment-ratio-or-equality` | 线段比或等量关系 | 证明或求线段等量、比例、倍分关系 |
| `folding-ratio-expression` | 折叠定值与表达式 | 折纸产生全等、相似、比例定值或代数表达 |

## 解题方法 (Method)

---

### method: known-root-factorization

**名称**：已知根分解抛物线

**触发条件**：已知抛物线与 `x` 轴交于某个固定点，或已知 `A(-1,0)` 在抛物线上。

**初中允许方法**：
- 先由已知根写出因式结构，如 `y=a(x+1)(x-r)`。
- 结合系数关系或另一个点坐标消去未知量。
- 用根与截距读出坐标关系。

**禁用方法**：
- 明明已有根，却再引入无必要的新参数。
- 在早期步骤提前代入后面才求出的最终系数。

**标准推导模板**：
1. 由已知点在 `x` 轴上确定一个根。
2. 写出分解式或由代入化简出通式。
3. 读出另一个交点、截距或系数关系。

**图形要求**：
- 只标当前已确定的根或符号坐标。
- 未求出系数前，不标最终数值坐标。

**常见错误**：
- 把可直接由根读出的点改写成新未知点，增加学生负担。

**构造宏状态**：不适用。

### method: coefficient-from-point-on-parabola

**名称**：点在抛物线上求系数

**触发条件**：题目给出点在抛物线上，需要求 `a`、`b`、`c` 或参数。

**初中允许方法**：
- 先把已知点代入解析式。
- 若有几何条件，先用几何方法求出点坐标，再代入抛物线。
- 每个点代入单独成行，避免跳步。

**禁用方法**：
- 未说明点坐标来源就直接代入。
- 把待求系数做成主滑动参数。

**标准推导模板**：
1. 由已知点代入，化简解析式。
2. 用几何条件求出新点坐标。
3. 将新点坐标代入抛物线。
4. 结合取值条件筛选答案。

**图形要求**：
- 坐标标签使用当前代数状态。
- 系数求出前，保持符号坐标。

**常见错误**：
- 先列旋转候选点，再用代数筛选，缺少初中几何解释。

**构造宏状态**：不适用。

---

### method: axis-parallel-segment-coordinate

**名称**：轴平行线段转坐标

**触发条件**：题目给出水平或竖直线段关系，如 `NI`、`MH`、`MP`，可由已知点和线段长直接读出目标点坐标，再代入抛物线求参数。

**初中允许方法**：
- 先用竖直或水平距离读出线段长，如同横坐标下 `NI=3-yI`。
- 由线段关系求出另一条轴平行线段，再把它转成点坐标，如 `MH=15a-1` 得 `H(15a+2,5)`。
- 将得到的点坐标直接代入抛物线，并观察公因式、平方差或已知结构。

**禁用方法**：
- 为了完整表达交点，提前求出后续不用的根号式，如 `MH=√(16+2/a)-4`。
- 引入无必要中间变量后再消元；能直接写目标点坐标时，不绕到 `x_H`。
- 把轴平行距离题改成复杂根式方程或高次方程硬解。

**标准推导模板**：
1. 由已知点代入或系数关系化简抛物线。
2. 读出已知轴平行线段，例如 `NI=7a`。
3. 用题目线段关系求目标线段，例如 `MH=15a-1`。
4. 由已知点坐标直接写目标点坐标，例如 `H(15a+2,5)`。
5. 将目标点代入抛物线，提取公因式或用平方差求参数。

**图形要求**：
- 公共结论步骤只标后续真正复用的线段和点坐标。
- 在求参步骤再标目标线段和目标点坐标，如 `MH=15a-1`、`H(15a+2,5)`。
- 不在图中显示未使用的根号型交点或距离表达式。

**常见错误**：
- 把后续不用的 `MH` 根式放入公共结论，导致学生误以为要解根号方程。
- 公共步骤、求参步骤、导航标签和引用按钮的知识粒度不一致。

**构造宏状态**：不适用。

### method: right-triangle-congruence-coordinate

**名称**：直角三角形全等求坐标

**触发条件**：出现 `90°`、等长、旋转 90°、等腰直角，且需要求点坐标。

**初中允许方法**：
- 作垂线，形成两个直角三角形。
- 用全等或等腰直角关系转移边长。
- 由横纵距离读出坐标。

**禁用方法**：
- 以向量旋转矩阵作为学生主解。
- 只写候选坐标，不说明三角形关系。

**标准推导模板**：
1. 作未知点到坐标轴或已知直线的垂线。
2. 写出两个直角三角形。
3. 由角和边证明全等。
4. 转移腿长，读出未知点坐标。

**图形要求**：
- 必须画垂足、垂线和直角标记。
- 标出被转移的关键边长。

**常见错误**：
- 垂足没有出现，学生看不到坐标如何读出。

**构造宏状态**：观察中（2/3）。

---

### method: rotation-by-congruence

**名称**：旋转全等

**触发条件**：线段或图形绕某点旋转，题目要求旋转后点坐标、线段或系数。

**初中允许方法**：
- 用旋转保持长度和角度。
- 作垂足，将旋转关系转成直角三角形全等。
- 再由坐标轴读坐标。

**禁用方法**：
- 用旋转矩阵或向量公式作为主解。

**标准推导模板**：
1. 说明旋转得到的等长和垂直关系。
2. 作辅助垂线。
3. 证两个直角三角形全等。
4. 读坐标并代入需要的表达式。

**图形要求**：
- 画旋转前后的线段。
- 画用于读坐标的垂线和直角。

**常见错误**：
- 只画旋转后点，不画原线段和辅助三角形。

**构造宏状态**：观察中（2/3）。

### method: isosceles-right-triangle-transform

**名称**：等腰直角三角形转化线段

**触发条件**：目标式中有 `√2` 倍线段，或需要把水平/竖直距离转化为斜边。

**初中允许方法**：
- 构造等腰直角三角形，使目标线段成为斜边或直角边。
- 用 `斜边 = √2 · 直角边` 转化表达式。
- 结合 45° 角读出等长腿。

**禁用方法**：
- 用投影解释 `√2` 系数。
- 明明已由 `AN=√2·QN` 得到 `QN`，还额外作无必要垂足。

**标准推导模板**：
1. 说明构造目的。
2. 作等腰直角三角形。
3. 写出线段等量关系。
4. 将原目标式转化为折线和。

**图形要求**：
- 直角用 rightAngle 标记。
- 45° 只标真正的锐角。
- 不在图中重复结论区已有公式。

**常见错误**：
- 把直角误标成 `45°`。
- 坐标还未求出时提前标最终坐标。

**构造宏状态**：观察中（3/3，可讨论固化 `isoscelesRightTriangle`）。

### method: weighted-path-segment-transform

**名称**：含权重路径的线段转化

**触发条件**：目标式形如 `2DM+AM`、`√2MN+AN` 等带系数的距离和。

**初中允许方法**：
- 先提取公共因子或将权重写成可构造的线段。
- 用特殊直角三角形把 `1/2 AM`、`√2MN` 等变成图上的真实线段。
- 再进入将军饮马或折线拉直。

**禁用方法**：
- 直接展开距离平方或用解析几何最优化。
- 权重没有转化为图上线段就开始拉直。

**标准推导模板**：
1. 改写目标式，明确要构造的线段。
2. 构造特殊三角形。
3. 得到等长或倍长关系。
4. 转成普通折线最值。

**图形要求**：
- 权重对应的构造线段必须可见。
- 路径发现步骤不要塞入计算辅助点。

**常见错误**：
- 同一步同时做路径转化和最终计算，图形过载。

**构造宏状态**：观察中（2/3）。

### method: horse-drinking

**名称**：将军饮马

**触发条件**：折线和最小、动点在线或射线上运动、对称或拉直后可得到最短路径。

**初中允许方法**：
- 用对称点拉直折线。
- 用三点共线说明线段和最短。
- 若终点在固定直线或射线上，补充垂线段最短。

**禁用方法**：
- 点到直线距离公式作为主解。
- 向量投影作为主解。

**标准推导模板**：
1. 将目标转为折线路径。
2. 作对称点或辅助点。
3. 说明三点共线或垂直时取最短。
4. 用图形关系计算最短长度。

**图形要求**：
- 显示折线、拉直线段、最短状态。
- 公式放在推导区，图中少放公式卡片。

**常见错误**：
- 忘记检查最短点是否落在题目限定范围内。

**构造宏状态**：观察中（3/3，可讨论固化 `horsePathStraightening`）。

### method: hidden-circle-minimum

**名称**：隐圆最短模型

**触发条件**：动点满足固定线段所对直角，如 `∠OHB=90°`，或等价地动点在以固定线段为直径的圆上；目标是求动点到定点的距离最小值。

**初中允许方法**：
- 用直径所对圆周角是直角，识别动点轨迹为圆。
- 找圆心和半径。
- 连接定点与圆心，用 `定点到圆上点的最短距离 = 圆心距 − 半径`。
- 检查最短点是否落在题目限定的圆弧或象限内。

**禁用方法**：
- 直接设动点坐标并展开二次根式求最值作为主解。
- 只画一段弧，不说明完整圆轨迹来源。
- 忽略动点所在象限或圆弧限制。

**标准推导模板**：
1. 由直角条件识别隐圆。
2. 写出圆心、半径和动点所在圆弧。
3. 连接定点与圆心，说明最短时三点共线。
4. 计算圆心距减半径，并结合题设最小值反求参数。

**图形要求**：
- 画完整圆并轻填充，突出实际运动圆弧。
- 增加圆上动点局部控件，默认停在最短位置。
- 显示圆心、半径、定点到圆心线段、最短线段。
- 若相关点可在坐标轴两侧，必须分情况画出对应模型。

**常见错误**：
- 把 `∠OHB=90°` 误当作固定直角三角形，只画一个最短位置而不画轨迹。
- 漏掉 `C` 在 `y` 轴上半轴/下半轴等对称位置分支。

**构造宏状态**：观察中（1/3）。

### method: translation-path-transform

**名称**：平移路径转化

**触发条件**：目标式中两段线段可以通过平移变成同一起终点的折线。

**初中允许方法**：
- 将一条线段平移到等长线段。
- 把两段和转成折线和。
- 用两点之间线段最短求最小值。

**禁用方法**：
- 直接用坐标距离函数求极值。

**标准推导模板**：
1. 说明平移方向和距离。
2. 得到等长替换线段。
3. 折线拉直求最小。

**图形要求**：
- 原线段和平移后线段用颜色区分。
- 标出折线拉直后的直线。

**常见错误**：
- 平移后的端点没有定义清楚。

**构造宏状态**：观察中（1/3）。

### method: axis-symmetry-distance-difference

**名称**：轴对称距离差

**触发条件**：点在抛物线对称轴上，目标为距离差最大值，如 `PB-PM`。

**初中允许方法**：
- 先利用对称轴把一个距离换成到对称点的距离。
- 用三角形两边差小于第三边求最大值。
- 等号条件用三点共线说明。

**禁用方法**：
- 把两个距离全部展开成根式再求最值。

**标准推导模板**：
1. 判断两个点关于对称轴对称。
2. 将一个距离替换为等长距离。
3. 用三角形不等式确定最大值。
4. 由给定最大值反求参数。

**图形要求**：
- 标出对称轴和被替换的等长线段。
- 显示三点共线的取等条件。

**常见错误**：
- 只写不等式，没有说明对称替换来源。

**构造宏状态**：观察中（1/3）。

### method: tangent-definition-in-right-triangle

**名称**：直角三角形中的 tan 定义

**触发条件**：初中题出现角度和、45°、`tan`，需要求点坐标。

**初中允许方法**：
- 只用 `tan = 对边 / 邻边`。
- 借助已知 45° 角构造相等角。
- 把角度条件转成直角三角形边比。

**禁用方法**：
- 正切和差公式。
- 斜率夹角公式。

**标准推导模板**：
1. 找到已知 45° 或直角关系。
2. 作辅助交点，转化角度。
3. 在直角三角形中用 tan 定义求边。
4. 写直线方程或代入抛物线求点。

**图形要求**：
- 显示辅助点和对应直角三角形。
- 标出被使用的角。

**常见错误**：
- 用高中三角公式解释角度和。

**构造宏状态**：不适用。

### method: translation-overlap-area

**名称**：平移重叠面积

**触发条件**：图形平移，求重叠区域面积或范围。

**初中允许方法**：
- 按关键边与边界相交的阶段分段。
- 用大图形面积减去小三角形/梯形面积。
- 先求关键交点和范围，再写面积。

**禁用方法**：
- 不分阶段直接给面积公式。

**标准推导模板**：
1. 确定平移参数和关键边界。
2. 找出当前阶段的重叠多边形。
3. 分解面积。
4. 结合参数范围求取值范围。

**图形要求**：
- 高亮当前重叠区域。
- 标出决定阶段的交点。

**常见错误**：
- 面积公式对应的阶段范围写错。

**构造宏状态**：已由重叠渲染器支持，不新增宏。

### method: folding-overlap-area

**名称**：折叠重叠面积

**触发条件**：图形折叠后与原图或固定图形重叠，求面积或范围。

**初中允许方法**：
- 利用折叠保距和对称。
- 识别重叠多边形阶段。
- 用分割补形求面积。

**禁用方法**：
- 忽略折叠点的保距关系。

**标准推导模板**：
1. 写出折叠对应点和保距关系。
2. 判断重叠区域形状。
3. 分解面积并列式。
4. 求参数范围或面积范围。

**图形要求**：
- 显示折叠前后对应点。
- 高亮重叠区域。

**常见错误**：
- 把折叠后的边当作平移边处理。

**构造宏状态**：已由折叠/重叠渲染器支持，不新增宏。

---

### method: rotation-overlap-area

**名称**：旋转重叠面积

**触发条件**：旋转图形与固定图形重叠，求面积或范围。

**初中允许方法**：
- 使用旋转保距、保角。
- 找关键交点和阶段边界。
- 用三角形、四边形面积分解。

**禁用方法**：
- 不说明旋转关系直接写交点坐标。

**标准推导模板**：
1. 明确旋转中心、角度、对应点。
2. 求关键交点。
3. 识别重叠区域。
4. 分段求面积。

**图形要求**：
- 显示旋转后图形和重叠区域。
- 标出决定面积表达式的线段。

**常见错误**：
- 忽略旋转后图形和原图的对应关系。

**构造宏状态**：已由旋转/重叠渲染器支持，不新增宏。

### method: area-piecewise-by-overlap

**名称**：由重叠形状分段求面积

**触发条件**：动图形与固定图形的重叠区域随参数变化。

**初中允许方法**：
- 先列阶段，再列公式。
- 每段只画当前形状和必要辅助线。
- 面积范围由阶段端点或单调性判断。

**禁用方法**：
- 混用不同阶段的公式。

**标准推导模板**：
1. 找临界位置。
2. 画或说明当前阶段重叠形状。
3. 用分割补形写面积。
4. 求范围。

**图形要求**：
- 各阶段缩略图或步骤图要与公式一致。

**常见错误**：
- 公式正确但阶段端点开闭错误。

**构造宏状态**：不适用。

### method: parameter-range-by-sign-and-order

**名称**：由符号和大小关系求参数范围

**触发条件**：给出两个函数值同号、大小关系、恒成立条件，求参数范围。

**初中允许方法**：
- 先化简表达式。
- 用符号、开口方向、零点、大小比较分情况。
- 将不等式解集与参数条件相交。

**禁用方法**：
- 未分情况直接乘除含参数符号的式子。

**标准推导模板**：
1. 写出目标表达式。
2. 找零点或临界值。
3. 按符号/大小关系分类。
4. 合并范围。

**图形要求**：
- 可用数轴或函数示意辅助，不强求复杂几何图。

**常见错误**：
- 忽略端点是否能取等。

**构造宏状态**：不适用。

### method: vertical-distance-absolute-value

**名称**：铅直距离绝对值

**触发条件**：同一竖直线与抛物线、直线相交，两点距离为 `m`。

**初中允许方法**：
- 用同横坐标下的纵坐标差表示距离。
- 取绝对值得到分段或范围。
- 结合给定 `t` 范围求最大值或恒成立条件。

**禁用方法**：
- 忽略绝对值直接求范围。

**标准推导模板**：
1. 写出两个交点的纵坐标。
2. 作差并取绝对值。
3. 在给定范围内判断最大值或零点。
4. 反求参数范围。

**图形要求**：
- 显示竖直线和上下两个交点。
- 标出铅直距离。

**常见错误**：
- 把有向差当作距离。

**构造宏状态**：不适用。

### method: folding-congruence-relations

**名称**：折叠全等与等量关系

**触发条件**：折纸题中出现折痕、点的像、折后线段或角度关系。

**初中允许方法**：
- 利用折叠前后图形全等。
- 由折叠得到对应边相等、对应角相等。
- 折痕是对应点连线的垂直平分线。

**禁用方法**：
- 只凭图形直观断言线段相等或角平分。

**标准推导模板**：
1. 明确折叠前后的对应点。
2. 写出对应三角形全等。
3. 得到对应边、对应角相等。
4. 与正方形、矩形等基本性质合并推出目标关系。

**图形要求**：
- 显示折痕和折叠前后对应点。
- 关键等长线段或等角要在图中可读。

**常见错误**：
- 折叠像点、原点、折痕交点混用。

**构造宏状态**：观察中（2/3）。

### method: perpendicular-bisector-from-fold

**名称**：由折痕得到中垂线

**触发条件**：折纸对折或将一点折到另一点/某线上，需要证明某点到两个端点距离相等。

**初中允许方法**：
- 对折得到两个端点到折痕上的点距离相等。
- 说明折痕垂直并平分原线段，因此是中垂线。
- 用中垂线性质得到等距。

**禁用方法**：
- 把“在折痕上”直接说成“距离相等”，但不说明中垂线。

**标准推导模板**：
1. 由对折得到折痕垂直平分目标线段。
2. 说明目标点在折痕上。
3. 得到到两端点距离相等。

**图形要求**：
- 标出折痕、被垂直平分的线段和目标点。

**常见错误**：
- 忽略折痕必须是对应点连线的中垂线。

**构造宏状态**：观察中（1/3）。

### method: similar-triangle-ratio-transfer

**名称**：相似三角形比例传递

**触发条件**：题目要求证明定比，或通过两组三角形相似把一个比例转移到另一组线段。

**初中允许方法**：
- 用 AA 或 SAS 证明相似。
- 先得到一个比例，再重组成另一组相似条件。
- 通过相似三角形对应边得到目标比值。

**禁用方法**：
- 直接写比例结论，不说明相似三角形来源。

**标准推导模板**：
1. 证明第一组三角形相似，得到比例。
2. 将比例重组。
3. 结合夹角相等或另一组角相等，证明第二组三角形相似。
4. 由对应边得到目标比。

**图形要求**：
- 两组三角形分别用不同颜色或逐步高亮。

**常见错误**：
- 同侧/跨侧三角形对应关系混乱。

**构造宏状态**：观察中（2/3）。

### method: pythagorean-algebraic-expression

**名称**：勾股定理建立代数表达式

**触发条件**：已知直角三角形边长含参数，需要求另一个参数或代数式。

**初中允许方法**：
- 先用几何关系表示三边。
- 在直角三角形中列勾股方程。
- 化简得到参数表达式。

**禁用方法**：
- 没有说明直角来源就列勾股。

**标准推导模板**：
1. 确定直角三角形和直角。
2. 用参数表示三边。
3. 列勾股方程。
4. 化简求参数或目标表达式。

**图形要求**：
- 标出直角三角形三边，参数线段标签保持一致。

**常见错误**：
- 把折叠后的线段和原线段长度关系写反。

**构造宏状态**：不适用。

### method: parallel-line-segment-ratio

**名称**：平行线分线段成比例

**触发条件**：需要在平行边之间用线段比例求交点长度或坐标。

**初中允许方法**：
- 先说明两线平行。
- 用相似三角形或平行线分线段成比例。
- 代入已知长度求目标线段。

**禁用方法**：
- 不说明相似/平行来源就直接写比例。

**标准推导模板**：
1. 找到平行线。
2. 写出对应相似三角形或比例。
3. 代入长度。
4. 化简目标线段。

**图形要求**：
- 标出平行线和被截线。

**常见错误**：
- 分母使用全长还是部分长混淆。

**构造宏状态**：不适用。

### method: angle-doubling-isosceles-construction

**名称**：倍角条件的等腰构造

**触发条件**：出现 `∠CAB=2∠ABC` 这类倍角条件，需要求交点或参数。

**初中允许方法**：
- 作对称点或辅助点，把倍角拆成等腰三角形的外角/底角关系。
- 通过等腰三角形得到线段等长。
- 将几何长度转为坐标或系数方程。

**禁用方法**：
- 使用三角函数倍角公式。
- 直接用斜率夹角公式求角度关系。

**标准推导模板**：
1. 设小角为 `θ`。
2. 作对称点或辅助点构造等腰三角形。
3. 用三角形内外角关系落实倍角。
4. 得到等长线段并求坐标/参数。

**图形要求**：
- 显示对称点或辅助点，标出等腰边和倍角来源。

**常见错误**：
- 倍角条件只写成代数角度，图形里没有对应构造。

**构造宏状态**：观察中（1/3）。

### method: equal-angle-reflection-line

**名称**：等角确定镜像直线

**触发条件**：点在某条射线上，给出 `∠ABM=∠ABC` 这类等角条件，需要确定动点所在直线。

**初中允许方法**：
- 利用公共边或坐标轴，构造关于轴或直线的镜像点。
- 将等角条件转成两条直线关于轴对称。
- 由两个点确定目标直线。

**禁用方法**：
- 直接用斜率角公式。

**标准推导模板**：
1. 找到等角共享的顶点和基准射线。
2. 构造镜像点。
3. 说明目标直线经过已知点和镜像点。
4. 写出直线表达式或求交点。

**图形要求**：
- 显示镜像点和目标直线，标出等角。

**常见错误**：
- 镜像点方向取反错误。

**构造宏状态**：观察中（1/3）。

### method: vertical-area-decomposition

**名称**：铅垂底分解面积

**触发条件**：三角形跨过 `y` 轴或某条竖直线，适合用竖直线段作公共底分成左右两个三角形。

**初中允许方法**：
- 选竖直线段作公共底。
- 左右两个点到竖直线的水平距离作高。
- 两个三角形面积相加。

**禁用方法**：
- 直接套坐标面积行列式作为主解。

**标准推导模板**：
1. 找竖直公共底。
2. 求底长。
3. 求左右两侧水平高。
4. 面积相加并列方程。

**图形要求**：
- 画竖直底和左右两个三角形，标出底长和水平高。

**常见错误**：
- 水平距离符号当成有向坐标使用。

**构造宏状态**：观察中（1/3）。
