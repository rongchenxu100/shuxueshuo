# 解题过程

## 第（I）问

### Step 1
- 标题：由 30°直角三角形求 A 点坐标
- 目标：求点 A 的坐标。
- 推导：
  - ∵ AB∥x 轴，∠OAB＝30°
  - ∴ OA 与 x 轴正方向夹角为 30°
  - ∵ AO＝6
  - ∴ xA＝6cos30°＝3√3
  - ∴ yA＝6sin30°＝3
  - ∴ A＝（3√3，3）
- 当前结论：A＝（3√3，3）

### Step 2
- 标题：由斜边和线段差求 B 点坐标
- 目标：求点 B 的坐标。
- 推导：
  - ∵ ∠OAB＝30°
  - ∵ cos30°＝AO/AB
  - ∴ AB＝AO/cos30°＝6/(√3/2)＝4√3
  - ∵ C 在 y 轴上，A＝（3√3，3）
  - ∴ AC＝3√3
  - ∴ BC＝AB－AC＝√3
  - ∵ AB∥x 轴
  - ∴ B 的纵坐标为 3，横坐标为 －√3
  - ∴ B＝（－√3，3）
- 当前结论：AB＝4√3，BC＝√3，B＝（－√3，3）

## 第（II）①问

### Step 1
- 标题：由平移与翻折写出 N、A' 坐标
- 目标：用 t 表示 N、A'。
- 推导：
  - ∵ l' 是 l 向左平移 t 个单位得到
  - ∴ l'：x＝3√3－t
  - ∵ N 在 AB 上
  - ∴ N＝（3√3－t，3）
  - ∵ A 与 A' 关于 l' 对称
  - ∴ A'＝（3√3－2t，3）
- 当前结论：N＝（3√3－t，3），A'＝（3√3－2t，3）

### Step 2
- 标题：由边界状态判断重叠图形
- 目标：描述各取值范围内的重叠图形，并确定四边形范围。
- 推导：
  - ∵ 当 A' 与 B 重合时，3√3－2t＝－√3
  - ∴ t＝2√3
  - ∵ 当 N 与 C 重合时，3√3－t＝0
  - ∴ t＝3√3
  - ∵ 0＜t≤2√3 时，△A'MN 完全落在 △ABO 内
  - ∴ 重叠部分为三角形
  - ∵ 2√3＜t＜3√3 时，A' 在 B 的左侧，A'M 与 OB 相交，N 仍在 C 的右侧
  - ∴ 重叠部分为四边形
  - ∵ t＝3√3 时，N 与 C 重合，四边形情形结束
- 当前结论：0＜t≤2√3 为三角形，2√3＜t＜3√3 为四边形

### Step 3
- 标题：由面积差求四边形面积
- 目标：求 S 关于 t 的表达式。
- 推导：
  - 设 A'M 与 OB 交于 P
  - ∵ 重叠四边形 BPMN＝△A'MN－△A'BP
  - ∵ A'N＝t，MN＝t/√3，且 A'N⊥MN
  - ∴ S△A'MN＝t²/(2√3)
  - ∵ A'B＝BP＝2t－4√3
  - ∴ △A'BP 是等腰三角形
  - 过 B 向 A'P 作垂线，垂足为 Q
  - ∵ ∠BA'P＝30°
  - ∴ BQ＝A'B·sin30°＝t－2√3
  - ∵ A'P＝2√3（t－2√3）
  - ∴ S△A'BP＝1/2·A'P·BQ＝√3（t－2√3）²
  - ∴ S＝t²/(2√3)－√3（t－2√3）²
  - ∴ S＝－(5/(2√3))t²＋12t－12√3
- 当前结论：S＝－(5/(2√3))t²＋12t－12√3

### Step 4
- 标题：合并本问答案
- 目标：写出四边形时的面积表达式和范围。
- 推导：
  - ∴ 当 2√3＜t＜3√3 时，
  - S＝－(5/(2√3))t²＋12t－12√3
- 当前结论：2√3＜t＜3√3，S＝－(5/(2√3))t²＋12t－12√3

## 第（II）②问

### Step 1
- 标题：由边界状态判断面积分段
- 目标：根据图形状态分段讨论 S。
- 推导：
  - ∵ 1/2≤t≤2√3 时，△A'MN 完全落在 △ABO 内
  - ∴ 重叠图形是三角形，面积随 t 增大而增大
  - ∵ 2√3＜t≤(11√3)/4 时，重叠图形为四边形
  - ∴ 四边形面积先增大后减小
  - ∴ 先按三角形段、四边形段分别求面积范围，再合并结果
- 当前结论：1/2≤t≤2√3：三角形，面积变大；2√3＜t≤11√3/4：四边形，面积先变大再变小

### Step 2
- 标题：在三角形段求面积范围
- 目标：求 1/2≤t≤2√3 内 S 的范围。
- 推导：
  - ∵ A'N＝t，MN＝t/√3
  - ∵ A'N⊥MN
  - ∴ S＝1/2·t·t/√3＝t²/(2√3)
  - ∵ 1/2≤t≤2√3
  - ∴ S 随 t 增大而增大
  - ∴ √3/24≤S≤2√3
- 当前结论：三角形段：√3/24≤S≤2√3

### Step 3
- 标题：在四边形段求最大值
- 目标：求 2√3＜t≤(11√3)/4 内 S 的范围。
- 推导：
  - ∵ S＝－(5/(2√3))t²＋12t－12√3
  - ∴ 该二次函数开口向下
  - ∵ 对称轴 t＝12√3/5
  - ∵ 2√3＜12√3/5＜11√3/4
  - ∴ 最大值在 t＝12√3/5 处取得
  - ∴ Smax＝12√3/5
  - ∵ 左端接三角形段，S＝2√3
  - ∵ t＝(11√3)/4 时，S＝67√3/32
  - ∴ 四边形段的面积范围为 2√3＜S≤12√3/5
- 当前结论：四边形段：2√3＜S≤12√3/5

### Step 4
- 标题：合并最终答案
- 目标：合并两个分段的面积范围。
- 推导：
  - ∵ 三角形段得到 √3/24≤S≤2√3
  - ∵ 四边形段得到 2√3＜S≤12√3/5
  - ∴ √3/24≤S≤12√3/5
- 当前结论：√3/24≤S≤12√3/5
