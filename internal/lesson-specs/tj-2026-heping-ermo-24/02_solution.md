# 解题过程

## 第（Ⅰ）问

### Step 1
- 标题：由 30°-60°-90° 三角形求 A 坐标
- 目标：求点 A 的坐标。
- 推导：
  - ∵ AB∥OC，OC 在 x 轴上
  - ∴ AB 为水平线段，A 与 B 的纵坐标相同
  - ∵ BC＝2，∠BCO＝60°
  - ∴ B 到 x 轴的高度为 2sin60°＝√3
  - ∴ A 的纵坐标也是 √3
  - ∵ OA＝2，O(0，0)，A 在第一象限
  - ∴ x_A²+(√3)²＝2²
  - ∴ x_A＝1
- 当前结论：A(1，√3)。

### Step 2
- 标题：由 BC 的方向求 B 坐标
- 目标：求点 B 的坐标。
- 推导：
  - ∵ C(5，0)，BC＝2，∠BCO＝60°
  - ∵ 从 C 指向 B 时，水平方向向左 2cos60°＝1，竖直方向向上 2sin60°＝√3
  - ∴ B(5－1，√3)
- 当前结论：B(4，√3)。

## 第（Ⅱ）①问

### Step 1
- 标题：由折痕平行关系表示 O′
- 目标：用 t 表示 P、Q、O′ 的位置。
- 推导：
  - ∵ OP＝t，P 在 OC 上
  - ∴ P(t，0)
  - ∵ l∥BC，BC 与 x 轴正方向夹角为 120°
  - ∴ l 的方向可看作与 x 轴正方向夹角为 120°
  - ∵ O 关于 l 对称后为 O′
  - ∴ PO′ 与 x 轴正方向夹角为 60°，且 PO′＝PO＝t
  - ∴ O′(t+tcos60°，tsin60°)
- 当前结论：P(t，0)，O′(3t/2，√3t/2)。

### Step 2
- 标题：由边界位置判断四边形范围
- 目标：确定重叠部分为四边形时 t 的范围。
- 推导：
  - ∵ O′P 与 BC 相交于点 E
  - ∵ 当 E 与 B 重合时，四边形重叠刚开始形成
  - ∴ 左边界为 t＝3
  - ∵ 当 P 与 C 重合、Q 与 B 重合时，重叠四边形退化
  - ∴ 右边界为 t＝5
  - ∴ 重叠部分为四边形时，3＜t＜5
- 当前结论：3＜t＜5。

### Step 3
- 标题：由两条 60° 斜线求 O′E
- 目标：用 t 表示线段 O′E。
- 推导：
  - ∵ O′P 与 x 轴正方向夹角为 60°
  - ∴ 直线 O′P 可写成 y＝√3(x－t)
  - ∵ BC 过 C(5，0)，且与 x 轴正方向夹角为 120°
  - ∴ 直线 BC 可写成 y＝－√3(x－5)
  - ∵ E 是 O′P 与 BC 的交点
  - ∴ √3(x_E－t)＝－√3(x_E－5)
  - ∴ x_E＝(t＋5)/2
  - ∵ E 在 O′P 上，O′P 与 x 轴夹角为 60°
  - ∴ O′E＝2t－5
- 当前结论：O′E＝2t－5，3＜t＜5。

## 第（Ⅱ）②问

### Step 1
- 标题：由折叠位置判断面积分段
- 目标：判断 3/2≤t≤9/2 内重叠面积的变化。
- 推导：
  - ∵ 3/2≤t≤2 时，折痕 l 交 OA 于点 R，折后重叠部分是三角形
  - ∴ 此时 S 随 t 增大而增大
  - ∵ 2≤t≤3 时，折后图形被梯形上边截成等高三角形，面积保持为 √3
  - ∴ 此时 S 取得最大值 √3
  - ∵ 3≤t≤9/2 时，重叠部分为四边形并逐渐缩小
  - ∴ 最小值只需看右端点 t＝9/2
- 当前结论：最大值为 √3，最小值候选为 t＝9/2。

### Step 2
- 标题：由三角形面积求左段端点
- 目标：求 t＝3/2 时的重叠面积，确认左端不是最小值。
- 推导：
  - ∵ 3/2≤t≤2 时，折痕与 OA 交于 R
  - ∵ 折叠前 △OPR 与折叠后重叠三角形全等
  - ∴ S＝S△OPR
  - ∵ OP＝t，R 到 x 轴的高为 √3t/2
  - ∴ S＝1/2·t·√3t/2＝√3t²/4
  - ∴ 当 t＝3/2 时，S＝9√3/16
- 当前结论：左端面积为 9√3/16。

### Step 3
- 标题：由等底等高判断最大值
- 目标：说明 2≤t≤3 时面积恒为 √3。
- 推导：
  - ∵ 2≤t≤3 时，折后重叠部分的底边在 AB 上，高为 √3
  - ∵ 折叠后的斜边被 AB 截出的水平长度为 2
  - ∴ S＝1/2·2·√3＝√3
  - ∴ S 的最大值为 √3
- 当前结论：S_max＝√3。

### Step 4
- 标题：由右端四边形求最小值
- 目标：求 t＝9/2 时的重叠面积。
- 推导：
  - ∵ 3≤t≤9/2 时，重叠四边形随 t 增大而缩小
  - ∴ 最小值在 t＝9/2 处取得
  - ∵ t＝9/2 时，P(9/2，0)，E(19/4，√3/4)，Q(7/2，√3)
  - ∵ B(4，√3)
  - ∴ 重叠四边形为 PEBQ
  - ∵ BQ＝1/2，点 P 到 AB 的距离为 √3
  - ∴ S△PBQ＝1/2·1/2·√3＝√3/4
  - ∵ PE＝1/2，BE＝3/2，∠PEB＝60°
  - ∴ S△PEB＝1/2·1/2·3/2·sin60°＝3√3/16
  - ∴ S＝S△PBQ＋S△PEB＝7√3/16
- 当前结论：S_min＝7√3/16。

### Step 5
- 标题：由最大最小值合并范围
- 目标：写出 S 的取值范围。
- 推导：
  - ∵ 在 3/2≤t≤2 时，S 从 9√3/16 增大到 √3
  - ∵ 在 2≤t≤3 时，S＝√3
  - ∵ 在 3≤t≤9/2 时，S 从 √3 减小到 7√3/16
  - ∴ S 的最小值为 7√3/16，最大值为 √3
- 当前结论：7√3/16≤S≤√3。
