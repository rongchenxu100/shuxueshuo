# 题目标准化

## 基本信息
- 题号：第 2 题
- 来源：2025部分区二模
- 题型：坐标几何 + 折叠 + 最值

## 题目原文
2．（2025部分区二模）将一个矩形纸片 ABCD 放置在平面直角坐标系中，O（0，0），A（-2，0），B（4，0），D（-2，2√3），点 C 在第一象限，CD 与 y 轴交于点 G，P 为线段 OB 上一点，过点 P 作直线 l 交 CD 于点 Q，∠APQ＝60°。沿直线 PQ 折叠该纸片，折叠后点 A，D 的对应点分别为 A'，D'。

（I）如图 Z4-21①，当点 P 与点 O 重合时，点 Q 与点 D 重合，则点 C 的坐标为______，A' 的坐标为______。

（II）设折叠后与矩形 OBCG 重叠部分的面积为 S。设 OP＝t。

① 如图 Z4-21②，当折叠后四边形 A'D'QP 与矩形 OBCG 重叠部分为五边形时，A'D' 与 CG 交于点 F，PQ 与 OG 交于点 E，用含 t 的式子表示线段 A'F 的长，并直接写出 t 的取值范围；

② 当 3/2≤t≤7/2 时，求 S 的取值范围（直接写出结果即可）。

## 已知条件
- O（0，0）
- A（-2，0）
- B（4，0）
- D（-2，2√3）
- ABCD 为矩形
- 点 C 在第一象限
- CD 与 y 轴交于点 G
- P 在线段 OB 上
- 过点 P 作直线 l，交 CD 于点 Q
- ∠APQ＝60°
- 沿直线 PQ 折叠
- A，D 的对应点分别为 A'，D'
- 设 OP＝t

## 几何对象

### 点
- O，A，B，C，D，G
- P，Q
- A'，D'
- E，F

### 线
- x 轴，y 轴
- OB，CD，CG，OG
- PQ（折痕）
- A'D'

### 区域
- 原矩形 ABCD
- 右侧矩形 OBCG
- 折叠后四边形 A'D'QP
- 折叠后与矩形 OBCG 的重叠部分

## 动态参数
- 参数名：t
- 含义：t＝OP
- 范围：0≤t≤4

## 小问列表
1. 当 P 与 O 重合时，求 C 与 A' 的坐标。
2. 当重叠部分为五边形时，求 A'F 与 t 的关系，并求 t 的范围。
3. 当 3/2≤t≤7/2 时，求重叠面积 S 的取值范围。

## 标准答案
- 第（I）问：C＝（4，2√3），A'＝（1，√3）
- 第（II）①问：A'F＝（2-t）√3，且 2/3<t<2
- 第（II）②问：23√3/8 ≤ S ≤ 4√3
