# 解题过程

## 第（I）问

### Step 1
- 标题：利用矩形性质求 C 点坐标
- 目标：求出点 C 的坐标。
- 推导：
  - ∵ ABCD 是矩形，且 A（-2，0），B（4，0），D（-2，2√3）
  - ∵ AB 与 x 轴重合，AD 与 y 轴平行
  - ∴ 矩形的右上角点 C 与点 B 的横坐标相同，与点 D 的纵坐标相同
  - ∴ C＝（4，2√3）
- 当前结论：C＝（4，2√3）

### Step 2
- 标题：构建直角三角形求 A' 坐标
- 目标：求出点 A 关于折痕 OD 的对应点 A'。
- 推导：
  - ∵ P 与 O 重合，Q 与 D 重合
  - ∵ ∠APQ＝60°
  - ∵ 点 A 关于折痕 OD 的对应点是 A'，折叠前后对应角相等
  - ∴ ∠DOA'＝∠APQ＝60°
  - ∵ OA＝2，折叠前后对应距离相等
  - ∴ OA'＝OA＝2
  - ∵ 过点 A' 向 x 轴作垂线，垂足记为 H
  - ∴ ∠A'OH＝60°
  - ∴ 在 Rt△OA'H 中，∠HOA'＝60°
  - ∵ OA'＝2
  - ∴ OH＝2cos60°＝1
  - ∴ A'H＝2sin60°＝√3
  - ∴ A' 的横坐标是 1，纵坐标是 √3
  - ∴ A'＝（1，√3）
- 当前结论：A'＝（1，√3）

## 第（II）①问

### Step 1
- 标题：构建直角三角形求 Q 点坐标和 DQ
- 目标：把后面要用的长度先用 t 表示出来。
- 推导：
  - ∵ P＝（t，0）
  - ∵ 过点 Q 向 x 轴作垂线，垂足记为 M
  - ∵ Q 在 CD 上，所以 Q 的纵坐标与点 D 相同
  - ∴ QM＝2√3
  - ∵ 在 Rt△PQM 中，∠APQ＝60°，tan60°＝QM/PM
  - ∵ QM＝2√3
  - ∴ PM＝2
  - ∵ P＝（t，0），且 M 在 P 的左边 2 个单位
  - ∴ M＝（t-2，0）
  - ∴ Q＝（t-2，2√3）
  - ∵ G＝（0，2√3）
  - ∴ QG＝0-（t-2）＝2-t
  - ∵ DG＝2
  - ∴ DQ＝DG-QG＝2-（2-t）＝t
- 当前结论：Q＝（t-2，2√3），DQ＝t

### Step 2
- 标题：由折叠关系求 D'Q
- 目标：把折后对应线段也写成 t。
- 推导：
  - ∵ 点 D 关于折痕 PQ 的对应点是 D'
  - ∴ D'Q＝DQ＝t
- 当前结论：D'Q＝t

### Step 3
- 标题：由直角三角形求 D'F
- 目标：把 D'F 用 t 表示出来。
- 推导：
  - ∵ 点 D 关于折痕 PQ 的对应点是 D'
  - ∴ ∠D'＝∠D＝90°
  - ∵ ∠DQP＝∠D'QP＝120°
  - ∵ ∠FQP＝∠APQ＝60°
  - ∴ ∠D'QF＝120°-60°＝60°
  - ∵ 在 Rt△D'QF 中，tan60°＝D'F/D'Q
  - ∵ D'Q＝t
  - ∴ D'F＝t√3
- 当前结论：D'F＝t√3

### Step 4
- 标题：由线段和差求 A'F
- 目标：求出 A'F 与 t 的关系。
- 推导：
  - ∵ 折叠前后对应边长相等，且 AD＝2√3
  - ∴ A'D'＝AD＝2√3
  - ∴ A'F＝A'D'-D'F
  - ∵ A'D'＝2√3，D'F＝t√3
  - ∴ A'F＝2√3-t√3＝（2-t）√3
- 当前结论：A'F＝（2-t）√3

### Step 5
- 标题：由边界位置判断五边形出现和结束的状态
- 目标：判断五边形情形从哪里开始、到哪里结束。
- 推导：
  - ∵ 当 A'D' 与 CG 刚好交于点 G 时，重叠部分还是四边形
  - ∵ 点 P 继续向右移动后，重叠部分变成五边形
  - ∵ 当 Q 与 G 重合时，A' 与 C 重合，此时重叠部分变成三角形
  - ∵ 点 P 再继续向右移动后，重叠部分变成四边形
  - ∴ 左边界：A'D' 与 CG 交于点 G
  - ∴ 右边界：Q 与 G 重合
- 当前结论：由左边界和右边界求 t 的范围

### Step 6
- 标题：由左边界求 t 的下界
- 目标：求出 A'D' 与 CG 交于 G 时的 t。
- 推导：
  - ∵ 左边界时，F 与 G 重合
  - ∵ 在 Rt△D'QF 中，∠D'QF＝60°，D'Q＝t
  - ∴ cos60°＝D'Q/QF＝t/QF
  - ∴ QF＝2t
  - ∵ Q＝（t-2，2√3），G＝（0，2√3）
  - ∴ QG＝0-（t-2）＝2-t
  - ∵ 左边界时，F 与 G 重合
  - ∴ QF＝QG
  - ∴ 2t＝2-t
  - ∴ t＝2/3
- 当前结论：左边界对应 t＝2/3

### Step 7
- 标题：由右边界求 t 的上界
- 目标：求出 Q 与 G 重合时的 t。
- 推导：
  - ∵ 右边界时，Q 与 G 重合
  - ∵ Q＝（t-2，2√3）
  - ∵ G＝（0，2√3）
  - ∴ t-2＝0
  - ∴ t＝2
- 当前结论：右边界对应 t＝2

### Step 8
- 标题：合并最终答案
- 目标：把表达式和范围一起收束。
- 推导：
  - ∴ A'F＝（2-t）√3，且 2/3<t<2
- 当前结论：A'F＝（2-t）√3，2/3<t<2

## 第（II）②问

### Step 1
- 标题：由图形变化判断面积分段
- 目标：先判断各种图形情况，再确定最大值和最小值可能出现在哪里。
- 推导：
  - ∵ 当 3/2≤t<2 时，重叠部分是五边形
  - ∵ 当 t＝2 时，重叠部分是三角形，此时 Q 与 G 重合，A' 与 C 重合
  - ∵ 当 2<t≤7/2 时，重叠部分是四边形
  - ∵ 观察图形可知，当 t 从 3/2 变化到 2 时，重叠部分面积逐渐变大
  - ∵ 当 t 从 2 变化到 7/2 时，重叠部分面积逐渐变小
  - ∴ 当 t＝2 时，面积取得最大值
  - ∴ 最小值只需比较 t＝3/2 和 t＝7/2 时的面积
- 当前结论：最大值在 t＝2 时取得，最小值只需比较 t＝3/2 和 t＝7/2

### Step 2
- 标题：当 t＝2 时由三角形面积求最大值
- 目标：求出 t＝2 时的面积。
- 推导：
  - ∵ t＝2 时，重叠部分是三角形
  - ∵ GC＝4，点 P 到直线 CG 的距离是 2√3
  - ∴ S＝1/2×4×2√3＝4√3
- 当前结论：当 t＝2 时，S＝4√3

### Step 3
- 标题：当 t＝3/2 时由面积分割求最小值候选
- 目标：求出 t＝3/2 时的面积。
- 推导：
  - ∵ 重叠部分面积
    ＝S梯形QPA'D'-S△QGE-S△QFD'
  - ∵ t＝3/2
  - ∵ 折叠前后对应图形面积相等
  - ∴ S梯形QPA'D'＝S梯形DQPA
  - ∵ DQ＝t，AP＝2+t，AD＝2√3
  - ∴ S梯形DQPA＝1/2×（DQ+AP）×AD
  - ∴ S梯形DQPA＝1/2×（t+2+t）×2√3＝（2t+2）√3
  - ∴ 当 t＝3/2 时，S梯形QPA'D'＝4√3
  - ∵ 由第（II）①问已得 DQ＝t
  - ∴ 当 t＝3/2 时，QG＝DG-DQ＝2-3/2＝1/2
  - ∵ 点 E 在 OG 上，且 QE 与 x 轴成 60° 角
  - ∴ 在 Rt△QGE 中，tan60°＝GE/QG
  - ∴ GE＝QG·tan60°＝1/2×√3＝√3/2
  - ∴ S△QGE＝1/2×QG×GE＝1/2×1/2×√3/2＝√3/8
  - ∵ 由第（II）①问已得 D'Q＝t，QF＝2t
  - ∴ 当 t＝3/2 时，D'Q＝3/2，QF＝3
  - ∵ 在 Rt△QFD' 中，∠FQD'＝60°
  - ∴ S△QFD'＝1/2×D'Q×QF×sin60°
  - ∴ S△QFD'＝1/2×3/2×3×√3/2＝9√3/8
  - ∴ S＝4√3-√3/8-9√3/8＝15√3/4
- 当前结论：当 t＝3/2 时，S＝15√3/4

### Step 4
- 标题：当 t＝7/2 时由四边形面积求最小值候选
- 目标：求出 t＝7/2 时的面积。
- 推导：
  - ∵ t＝7/2 时，重叠部分是四边形
  - ∵ 直线 PA' 交 BC 于点 R，交 CG 的延长线于点 T
  - ∴ S＝S△TQP-S△TCR
  - ∵ t＝7/2，Q＝（3/2，2√3）
  - ∵ PA' 与 x 轴成 60° 角，T 在直线 CG 上
  - ∴ 点 T 到 x 轴的距离是 2√3
  - ∵ 在 Rt△PTN 中，∠TPN＝60°，TN＝2√3（其中 N 为点 T 在 x 轴上的垂足）
  - ∴ tan60°＝TN/PN＝2√3/PN
  - ∴ PN＝2
  - ∵ P 的横坐标是 7/2，且点 T 在点 P 的右边
  - ∴ 点 T 的横坐标是 7/2＋2＝11/2
  - ∴ QT＝11/2-3/2＝4
  - ∴ S△TQP＝1/2×4×2√3＝4√3
  - ∵ CT＝11/2-4＝3/2
  - ∵ 在 Rt△TCR 中，∠T＝60°
  - ∴ CR＝CT·tan60°＝3√3/2
  - ∴ S△TCR＝1/2×3/2×3√3/2＝9√3/8
  - ∴ S＝S△TQP-S△TCR
  - ∴ S＝4√3-9√3/8＝23√3/8
- 当前结论：当 t＝7/2 时，S＝23√3/8

### Step 5
- 标题：合并最终答案
- 目标：把两段结果合并成范围。
- 推导：
  - ∵ Step 1 已知最大值在 t＝2 时取得
  - ∵ 当 t＝2 时，S＝4√3
  - ∵ Step 1 已知最小值只需比较 t＝3/2 和 t＝7/2
  - ∵ 当 t＝3/2 时，S＝15√3/4；当 t＝7/2 时，S＝23√3/8
  - ∴ 最大值是 4√3
  - ∴ 最小值是 23√3/8
  - ∴ 23√3/8 ≤ S ≤ 4√3
- 当前结论：23√3/8 ≤ S ≤ 4√3
