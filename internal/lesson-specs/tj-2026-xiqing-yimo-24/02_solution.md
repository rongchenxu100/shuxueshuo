# 解题过程

## 第（1）问

### Step 1
- 标题：利用等边三角形求 A 点坐标
- 目标：求点 A 的坐标。
- 推导：
  - ∵ O＝（0，0），B＝（6√3，0）
  - ∴ OB＝6√3
  - ∵ △OAB 是等边三角形，A 在第一象限
  - ∴ A 的横坐标为 OB 的一半，即 3√3
  - ∴ A 的纵坐标为 (6√3)×√3/2＝9
  - ∴ A＝（3√3，9）
- 当前结论：A＝（3√3，9）

### Step 2
- 标题：由 30° 直角三角形求 C 点坐标
- 目标：求点 C 的坐标。
- 推导：
  - ∵ D＝（－6√3，0），O＝（0，0）
  - ∴ OD＝6√3
  - ∵ ∠CDO＝90°，∠COD＝30°
  - ∴ CD＝OD·tan30°＝6√3×√3/3＝6
  - ∵ C 在第二象限，CD⊥x 轴
  - ∴ C＝（－6√3，6）
- 当前结论：C＝（－6√3，6）

## 第（2）①问

### Step 1
- 标题：在 Rt△OMO'、Rt△AMN 中求 AN
- 目标：用几何直角三角形表示 AN，不写直线解析式联立。
- 推导：
  - ∵ ∠AOB＝60°，∠C'O'D'=30°
  - ∴ ∠OMO'＝90°，△OMO' 是直角三角形
  - ∵ OO'＝t
  - ∴ OM＝t／2
  - ∵ OA＝6√3（等边 △OAB 边长）
  - ∴ AM＝OA－OM＝6√3－t／2
  - ∵ Rt△AMN 中，∠MAN＝∠OAB＝60°
  - ∴ AN＝AM／cos60°＝2·AM＝12√3－t
- 当前结论：
  - OM＝t／2
  - AM＝6√3－t／2
  - AN＝2·AM＝12√3－t

### Step 2
- 标题：判断五边形情形下 t 的范围
- 目标：只用两端临界位置写出不等式，右边界用「C' 在 OA 上」判定，不引入 P 点。
- 推导：
  - ∵ t＝6√3 时，D' 与 O 重合，重叠部分是三角形
  - ∵ 6√3＜t＜8√3 时，重叠部分是五边形
  - ∵ t＝8√3 时，点 C' 落在 OA 上，重叠部分是四边形
  - ∴ 重叠部分为五边形时，6√3＜t＜8√3
- 当前结论：6√3＜t＜8√3

### Step 3
- 标题：合并本问答案
- 目标：写出 AN 与 t 的范围。
- 推导：
  - ∴ AN＝12√3－t
  - ∴ 6√3＜t＜8√3
- 当前结论：AN＝12√3－t，6√3＜t＜8√3

## 第（2）②问

### Step 1
- 标题：分四种情形判断重叠图形与极值位置
- 目标：在 6√3≤t≤10√3 上按端点与区间划分四种情况，判断 S 的最大值、最小值分别应在哪一段求。
- 推导：
  - **（1）t＝6√3**：D' 与 O 重合，重叠部分退化为三角形，得左端面积 S(6√3)＝27√3/2
  - **（2）6√3＜t＜8√3**：重叠部分为五边形 D'PMNB，S 表为开口向下的二次函数，**这一段上的最大值必在区间内部取得**
  - **（3）8√3≤t＜10√3**：重叠部分为四边形 D'C'NB，S 随 t 增大而**单调递减**
  - **（4）t＝10√3**：平移三角形即将离开 △OAB，S＝S(10√3)＝6√3，为 **6√3≤t≤10√3 上的最小值**
  - ∵ （2）给出全范围内的最大候选，（3）（4）表明右段递减且右端更小
  - ∴ **S 的最大值在 6√3＜t＜8√3 内取得**；**最小值在 t＝10√3 取得**
- 当前结论：下一步用五边形情形的 S（关于 t）求最大值；最小值在 t＝10√3 用重叠三角形面积求

### Step 2
- 标题：在6√3＜t＜8√3 中由五边形面积求 S 的最大值
- 目标：用 **S＝S△OMO'－S△D'PO－S△NBO'** 表出 S(t)：三块分别由 **Rt△OMO'**、**Rt△D'PO**（即 **∠OD'P＝90°** 的 **Rt△OPD'**）、**等腰△NBO'** 求面积。
- 推导：
  - ∵ 6√3＜t＜8√3 时重叠五边形 D'PMNB，**S＝S△OMO'－S△D'PO－S△NBO'**
  - **Rt△OMO'**（∠OMO'＝90°）：平移保角 **∠C'O'D'＝30°**，故 **∠MO'O＝30°**；**M** 在 **OA** 上，**O'** 在 x 轴正向，**∠MOO'** 即 **OA** 与 x 轴正向夹角，等于 **∠AOB＝60°**；**OO'＝t**。∴ **OM＝OO'·cos60°＝t/2**，**O'M＝OO'·sin60°＝t√3/2**，**S△OMO'＝½·OM·O'M＝√3 t²/8**
  - **Rt△D'PO**（∠OD'P＝90°）：**D'** 在 x 轴，**P** 在 **OA** 上，故 **∠POD'** 即 **OA** 与 x 轴夹角 **60°**；**OD'＝t－6√3**。∴ **D'P＝OD'·tan60°**＝**√3 t－18**，**S△D'PO＝½·OD'·D'P＝√3 t²/2－18t＋54√3**
  - **等腰△NBO'**：**BO'＝OO'－OB＝t－6√3**；**N** 在 **C'O'** 上，**∠BO'N＝∠C'O'D'＝30°**；**N** 在 **AB** 上，**∠NBO'＝180°－∠ABO＝120°**，∴ **∠BNO'＝30°＝∠BO'N**，**BN＝BO'**。**S△NBO'＝½·BN·BO'·sin120°＝√3/4·(t－6√3)²＝√3 t²/4－9t＋27√3**
  - ∴ S＝√3 t²/8－(√3 t²/2－18t＋54√3)－(√3 t²/4－9t＋27√3)＝**－5√3 t²/8＋27t－81√3**
  - ∵ 二次项系数－5√3/8＜0，开口向下，对称轴 t＝36√3/5，且 6√3＜36√3/5＜8√3
  - ∴ S_max＝81√3/5
  - ∵ 端点 S(6√3)＝27√3/2、S(8√3)＝15√3 均小于 81√3/5
- 当前结论：S_max＝81√3/5

### Step 3
- 标题：在 t＝10√3 时用重叠三角形面积求 S 的最小值
- 目标：用 **BO'＝OO'－OB**、等腰 **△NBO'** 与 **Rt△NBD'**（∠ND'B＝90°）求面积，不写坐标差。
- 推导：
  - ∵ t＝10√3 时 C' 与 N 重合，重叠部分为 **Rt△NBD'**（**∠ND'B＝90°**，即 **D'N⊥x 轴**，D'、B 在 x 轴上）
  - **求 BO'**：在 x 轴上 **OO'＝t**，**OB＝6√3**，且 O、B、O' 同线且 O' 在 B 右侧，∴ **BO'＝OO'－OB＝t－6√3**；代入 t＝10√3 得 **BO'＝4√3**
  - **△NBO' 为等腰**：平移后 **∠C'O'D'＝30°**，N 在 **C'O'** 上，故 **∠BO'N＝30°**；**N 在 AB 上**，**∠ABO＝60°**，又 **BO 与 BO' 在直线 OB 上反向**，∴ **∠NBO'＝180°－∠ABO＝120°**；由三角形内角和，**∠BNO'＝180°－30°－120°＝30°**＝**∠BO'N**，∴ **BN＝BO'＝4√3**
  - **Rt△NBD'**：**∠ND'B＝90°**，**∠NBD'＝60°**（**NB** 与 **BD'** 的夹角），斜边 **BN＝4√3**，∴ **D'B＝BN·cos60°＝2√3**，**D'N＝BN·sin60°＝6**
  - ∴ **S＝½·D'B·D'N＝½×2√3×6＝6√3**
- 当前结论：S_min＝6√3

### Step 4
- 标题：合并最终答案
- 目标：合并两段得到 S 的取值范围。
- 推导：
  - ∵ 左段最大值为 81√3/5
  - ∵ 右段最小值为 6√3
  - ∴ 当 6√3≤t≤10√3 时，6√3≤S≤81√3/5
- 当前结论：6√3≤S≤81√3/5
