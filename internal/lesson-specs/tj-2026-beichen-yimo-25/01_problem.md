# 题目原文

已知抛物线 \(y=\frac14x^2+bx-4\)（\(b\) 为常数）的顶点为 \(D\)，与 \(x\) 轴相交于点 \(A(-2,0)\) 和点 \(B\)，与 \(y\) 轴相交于点 \(C\)。

（Ⅰ）求该抛物线的解析式和顶点 \(D\) 的坐标；

（Ⅱ）点 \(P\) 为对称轴上一点，连接 \(BP\)，将线段 \(BP\) 绕点 \(P\) 逆时针旋转 \(90^\circ\)，使点 \(B\) 的对应点 \(Q\) 恰好落在抛物线上，求此时点 \(P\) 的坐标；

（Ⅲ）在线段 \(BC\) 上，是否存在一点 \(H\)，使 \(2AH+\sqrt2BH\) 的值最小？若存在，求出点 \(H\) 的坐标，并求出最小值；若不存在，请说明理由。

# 已知条件

- 抛物线：\(y=\frac14x^2+bx-4\)。
- 点 \(A(-2,0)\) 在抛物线上。
- \(B\) 是另一个 \(x\) 轴交点，\(C\) 是 \(y\) 轴交点。
- 点 \(P\) 在抛物线对称轴上。
- \(BP\) 绕点 \(P\) 逆时针旋转 \(90^\circ\) 后，点 \(B\) 对应到 \(Q\)，且 \(Q\) 在抛物线上。
- 点 \(H\) 在线段 \(BC\) 上。

# 小问列表

1. 求抛物线解析式和顶点 \(D\)。
2. 求满足旋转条件的点 \(P\)。
3. 判断并求 \(2AH+\sqrt2BH\) 的最小值及对应点 \(H\)。

# 标准答案

- （Ⅰ）抛物线解析式为 \(y=\frac14x^2-\frac32x-4\)，顶点 \(D(3,-\frac{25}{4})\)。
- （Ⅱ）\(P(3,-5)\) 或 \(P(3,9)\)。
- （Ⅲ）存在，\(H(4,-2)\)，最小值为 \(6\sqrt{10}\)。
