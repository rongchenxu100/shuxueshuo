# 可视化总体设计

本题没有原题配图，因此 `lesson-data.problem.lines` 保持文字题干，不设置 `originalFigures`。

全局视窗使用

```json
{ "minX": -2.2, "maxX": 7.2, "minY": -7.2, "maxY": 3.0 }
```

以同时容纳第（Ⅰ）问顶点、第（Ⅱ）①问的旋转点 \(D\)，以及第（Ⅱ）②问中 \(c=-13/2\) 时的最短路反射图。

# 第（Ⅰ）问

## `i1s1`：配方求顶点

- 显示固定抛物线 \(y=x^2-2x-3\)。
- 标出对称轴 \(x=1\) 与顶点 \(P(1,-4)\)。
- 图中只需要顶点和抛物线，不加入第（Ⅱ）中的 B、C、D。

# 第（Ⅱ）①问

## `q1s1`：由 A 是根确定 B、C

- 显示 \(a=2\)、由 \(A(-1,0)\) 化简得到的抛物线 \(y=(x+1)(2x+c)\)。
- 标出 \(A(-1,0)\)、\(B(-\frac c2,0)\)、\(C(0,c)\)。
- 这一图只呈现代数读出的未定坐标，不能提前出现 \(c=-\frac52\)、\(B(\frac54,0)\)、\(C(0,-\frac52)\) 或点 \(D\)。

## `q1s2`：作 DQ⊥x轴，用全等表示 D 的符号坐标

- 仍保持符号状态，不锁定最终 \(c\)。
- 连接 \(BC\)、\(BD\)，过 \(D\) 作 \(DQ\perp x\) 轴，垂足为 \(Q\)。
- 标出 \(B(-\frac c2,0)\)、\(C(0,c)\)、\(Q(\frac c2,0)\)、\(D(\frac c2,-\frac c2)\)。
- 显示 \(Rt△CBO\) 与 \(Rt△BDQ\)，标出两个直角。
- 用同色强调 \(BC\) 与 \(BD\) 表示等长关系。

## `q1s3`：代入抛物线求出 D

- 锁定 \(c=-\frac52\)，此时才显示 \(B(\frac54,0)\)、\(C(0,-\frac52)\)。
- 连接 \(BC\)、\(BD\)，标出 \(\angle CBD=90^\circ\) 和等长关系。
- 标出 \(D(-\frac54,\frac54)\)。

# 第（Ⅱ）②问

这一问用 `movingParam: "c"` 表示题目中的系数 \(c\)，默认锁定在最终值 \(-13/2\)。`expressionEnv` 中按顺序给出 `a=1`、`b=c+1`、对称轴与反射点坐标所需的表达式。

## `q2s1`：由 A 和 B 坐标化简函数表达式

- 显示动态抛物线 \(y=x^2+(c+1)x+c\)。
- 标出 \(A(-1,0)\)、\(B(-c,0)\)、\(C(0,c)\)。
- 显示对称轴 \(l\)。

## `q2s2`：角度条件确定 E 在 OC 上的位置

- 保留 \(A,B,C\) 和抛物线背景。
- 连接 \(BA,BC,BE\)。
- 标出 \(E(0,c/\sqrt3)\)。
- 用角弧表现 \(\angle ABE=30^\circ\)、\(\angle CBE=15^\circ\) 的二倍关系。

## `q2s3`：求 F

- 显示直线 \(BE\) 与对称轴 \(l\) 的交点 \(F\)。
- 标出 \(F\left(-\frac{c+1}{2},\frac{c-1}{2\sqrt3}\right)\)。
- 当前步骤强调对称轴和 \(BE\)，弱化角弧。

## `q2s4`：两次对称转化最短路

- 将局部视窗缩小到 \(A,B,E,F,A',F'\) 附近。
- 画出 \(A\) 关于 \(BE\) 的对称点 \(A'\)，以及 \(F\) 关于 \(BA\) 的对称点 \(F'\)。
- 保持 \(G\in BE\)、\(H\in BA\) 的示意，画出 \(A'G+GH+HF'\)。
- 用竖直线段 \(A'F'\) 表示最短距离。

## `q2s5`：由最小值求 c 和 E

- 锁定 \(c=-13/2\)。
- 标出 \(E(0,-13\sqrt3/6)\)。
- 显示最短线段 \(A'F'=5\sqrt3\)，并给出最终答案。
