# 解题过程

## 第（1）①问

### Step 1：化简函数表达式并求顶点

- 目标：把已知系数代入，再配方求顶点 \(P\)。
- 推导：
  - ∵ \(a=1,\ c=-5,\ 4a+b=0\)
  - ∴ \(b=-4\)
  - ∴ \(y=x^2-4x-5\)
  - ∴ \(y=(x-2)^2-9\)
  - ∴ 顶点 \(P(2,-9)\)
- 当前结论：
  - \(P(2,-9)\)

## 第（1）②问

### Step 1：由水平线求 \(D\)，由直线 \(BC\) 设 \(E\)

- 目标：先确定 \(D\)，再用一个参数表示 \(E\) 与 \(M\)。
- 推导：
  - ∵ \(y=x^2-4x-5\)
  - ∴ \(A(-1,0),\ B(5,0),\ C(0,-5)\)
  - ∵ 过 \(C\) 作 \(x\) 轴平行线与抛物线另交于 \(D\)
  - ∴ \(D(4,-5)\)
  - ∵ \(B(5,0),\ C(0,-5)\)
  - ∴ 直线 \(BC:\ y=x-5\)
  - 设 \(E(s,s-5)\)
  - ∵ \(\square CDME\) 是平行四边形
  - ∴ \(\overrightarrow{DM}=\overrightarrow{CE}=(s,s)\)
  - ∴ \(M(4+s,s-5)\)
- 当前结论：
  - \(D(4,-5)\)
  - \(E(s,s-5)\)
  - \(M(4+s,s-5)\)

### Step 2：点 \(M\) 在抛物线上求坐标

- 目标：把 \(M(4+s,s-5)\) 代入抛物线，求出非退化位置。
- 推导：
  - ∵ \(M(4+s,s-5)\) 在 \(y=x^2-4x-5\) 上
  - ∴ \(s-5=(4+s)^2-4(4+s)-5\)
  - ∴ \(s-5=s^2+4s-5\)
  - ∴ \(s^2+3s=0\)
  - ∴ \(s=0\) 或 \(s=-3\)
  - ∵ \(s=0\) 时 \(E=C,\ M=D\)，平行四边形退化
  - ∴ \(s=-3\)
  - ∴ \(M(1,-8)\)
- 当前结论：
  - \(M(1,-8)\)

## 第（2）问

### Step 1：由已知根和对称轴写出 \(B,C\)

- 目标：利用 \(A(-1,0)\) 与 \(4a+b=0\) 建立第（2）问的基本点位。
- 推导：
  - ∵ \(4a+b=0\)
  - ∴ \(b=-4a\)，抛物线对称轴为 \(x=2\)
  - ∵ \(A(-1,0)\) 是左侧交点
  - ∴ 另一个 \(x\) 轴交点为 \(B(5,0)\)
  - ∴ 抛物线可写为 \(y=a(x+1)(x-5)\)
  - ∴ \(C(0,-5a)\)
- 当前结论：
  - \(B(5,0)\)
  - \(C(0,-5a)\)
  - 对称轴为 \(x=2\)

### Step 2：由平行四边形 \(ACFG\) 表示 \(G\)

- 目标：把 \(F\) 与 \(G\) 的联动关系说清楚。
- 推导：
  - 设 \(F(2,u)\)
  - ∵ \(\square ACFG\) 是平行四边形
  - ∴ \(AG\parallel CF\)，且 \(AG=CF\)
  - ∵ \(A(-1,0),\ C(0,-5a),\ F(2,u)\)
  - ∴ 从 \(C\) 到 \(F\)：横向向右 \(2\)，纵向向上 \(u+5a\)
  - ∴ 从 \(A\) 到 \(G\) 也横向向右 \(2\)，纵向向上 \(u+5a\)
  - ∴ \(G(1,u+5a)\)
  - ∴ 当 \(F\) 在直线 \(x=2\) 上运动时，\(G\) 在直线 \(x=1\) 上同步运动
- 当前结论：
  - \(F(2,u)\)
  - \(G(1,u+5a)\)

### Step 3：构建平行四边形，将双动点问题转化为单动点问题

- 目标：把 \(BG\) 转化为从同一个动点 \(F\) 出发的线段。
- 推导：
  - 作平行四边形 \(BGFB_1\)
  - ∴ \(B_1F\parallel BG\)，且 \(B_1F=BG\)
  - ∵ 从 \(G\) 到 \(F\)：横向向右 \(1\)，纵向向下 \(5a\)
  - ∴ 从 \(B\) 到 \(B_1\) 也横向向右 \(1\)，纵向向下 \(5a\)
  - ∴ \(B_1(6,-5a)\)
  - ∴ \(BG=B_1F\)
  - ∴ \(BF+BG=BF+B_1F\)
  - ∴ 原来 \(F,G\) 两个动点的问题，转化为只研究动点 \(F\) 的折线
- 当前结论：
  - \(B_1(6,-5a)\)
  - \(BF+BG=BF+B_1F\)

### Step 4：反射拉直求最小值

- 目标：利用对称轴把折线 \(BF+B_1F\) 拉直。
- 推导：
  - ∵ \(A(-1,0), B(5,0)\) 关于对称轴 \(x=2\) 对称
  - ∵ \(F\) 在对称轴 \(x=2\) 上
  - ∴ \(BF=AF\)
  - ∴ \(BF+B_1F=AF+B_1F\)
  - ∵ \(AF+B_1F\ge AB_1\)
  - ∴ 当 \(A,F,B_1\) 三点共线时，\(BF+BG\) 最小
  - ∴ 最小值为 \(AB_1\)
- 当前结论：
  - \(BF+BG\) 的最小值为 \(AB_1\)

### Step 5：由最小值反求 \(a\)

- 目标：把 \(AB_1=7\sqrt2\) 写成关于 \(a\) 的方程。
- 推导：
  - ∵ \(A(-1,0),\ B_1(6,-5a)\)
  - ∴ \(AB_1^2=7^2+(5a)^2=49+25a^2\)
  - ∵ \(BF+BG\) 的最小值为 \(7\sqrt2\)
  - ∴ \(AB_1=7\sqrt2\)
  - ∴ \(49+25a^2=98\)
  - ∴ \(a^2=\dfrac{49}{25}\)
  - ∵ \(a>0\)
  - ∴ \(a=\dfrac75\)
- 当前结论：
  - \(a=\dfrac75\)
