# 解题过程

## 第（I）问

### Step 1
- 标题：由 30°-60° 关系求 C、B 坐标
- 目标：求点 C 和点 B 的坐标。
- 推导：
  - ∵ OC＝4√3，∠AOC＝30°
  - ∴ C 的横坐标为 4√3cos30°＝6，纵坐标为 4√3sin30°＝2√3
  - ∴ C（6，2√3）
  - ∵ OABC 是平行四边形，OA∥CB，AB∥OC
  - ∴ 从 C 到 B 的位移等于从 O 到 A 的位移
  - ∵ A（6，0）
  - ∴ B（12，2√3）
- 当前结论：C（6，2√3），B（12，2√3）。

## 第（II）①问

### Step 1
- 标题：由直角三角形求 AA′
- 目标：先求 AD，再由折叠关系得到 AA′。
- 推导：
  - ∵ A（6，0），M（t，0），且 t＞6
  - ∴ AM＝t－6
  - ∵ D 在 AB 上，AB∥OC，∠AOC＝30°
  - ∴ ∠DAM＝30°
  - ∵ M、A、O 在同一直线上，∠OMN＝60°
  - ∴ ∠AMN＝60°
  - ∴ △AMD 是直角三角形，且 ∠ADM＝90°
  - ∴ AD＝AM·sin60°＝√3(t－6)/2
  - ∵ 折痕 MN 是 AA′ 的垂直平分线，D 是 MN 与 AB 的交点
  - ∴ D 是 AA′ 的中点，AA′＝2AD＝√3(t－6)
- 当前结论：AD＝√3(t－6)/2，AA′＝√3(t－6)。

### Step 2
- 标题：由线段差求 A′B
- 目标：用含有 t 的式子表示 A′B。
- 推导：
  - ∵ AB∥OC，且 AB＝OC＝4√3
  - ∵ A′ 在线段 AB 上，且由上一步 AA′＝√3(t－6)
  - ∴ A′B＝AB－AA′＝4√3－√3(t－6)
  - ∴ A′B＝√3(10－t)
- 当前结论：A′B＝√3(10－t)。

### Step 3
- 标题：由边界位置判断五边形范围
- 目标：判断重叠部分为五边形时 t 的取值范围。
- 推导：
  - ∵ A′ 在线段 AB 上，左边界是 A′ 与 A 重合
  - ∴ t＞6 时五边形才可能真正形成
  - ∵ F 点为 MN 与 OC 的交点，OF＝√3t/2
  - ∵ OC＝4√3
  - ∴ OF＜OC，即 √3t/2＜4√3
  - ∴ t＜8
  - ∵ t＝6 时 A′ 与 D 都和 A 重合，t＝8 时 F 与 C 重合
  - ∴ 五边形阶段不含端点
- 当前结论：6＜t＜8。

## 第（II）②问

### Step 1
- 标题：由折痕位置判断面积分段
- 目标：把 4＜t＜10 内的重叠面积按图形阶段分类。
- 推导：
  - ∵ 4＜t≤6 时，折痕 MN 切 OA、OC，折后图形是三角形 O′MF，重叠部分是四边形 MFCE，面积逐渐变大
  - ∵ 6＜t＜8 时，折痕 MN 切 AB、OC，重叠部分是五边形 FCEA′D，面积先变大再变小
  - ∵ 8≤t＜10 时，折痕 MN 切 AB、BC，重叠部分是四边形，面积逐渐变小
  - ∴ 最大值只需看五边形阶段；最小值只会在 t→4+ 或 t→10− 的端点趋势中出现
- 当前结论：侧边四边形阶段给下界，五边形阶段给最大值。

### Step 2
- 标题：由左侧边界求最小值
- 目标：在左端边界直接代入，得到面积下界候选。
- 推导：
  - 取左端边界 t＝4 作候选
  - ∵ M（4，0），O′ 与 C 重合，F 是 OC 的中点
  - ∴ FC＝2√3，M 到直线 OC 的距离为 2
  - ∴ S＝S△MFC＝1/2×2√3×2＝2√3
  - ∵ 题设为 4＜t＜10
  - ∴ 左端下界 2√3 取不到
- 当前结论：左端给出严格下界候选 2√3。

### Step 3
- 标题：由三角形面积差求五边形面积最大值
- 目标：求 6＜t＜8 中 S 的最大值。
- 推导：
  - 在 6＜t＜8 中，重叠五边形为 FCEA′D
  - ∵ 可看作大三角形 △MFO′ 减去 △MDA′ 和 △CEO′
  - ∵ FO′＝√3t/2，M 到直线 OC 的距离为 t/2
  - ∴ S△MFO′＝√3t²/8
  - ∵ DA′＝AD＝√3(t－6)/2，MD＝(t－6)/2
  - ∴ S△MDA′＝√3(t－6)²/8
  - ∵ O′（3t/2，√3t/2），BC 的纵坐标为 2√3
  - ∴ O′ 到直线 BC 的距离为 √3(t－4)/2
  - 作：过 O′ 作 O′H⊥BC，垂足为 H
  - ∵ E 在 BC 上，A′O′ 与 BC 成 60°
  - ∴ ∠O′EH＝60°，△O′EH 是直角三角形
  - ∴ EH＝O′H/tan60°＝[√3(t－4)/2]/√3＝(t－4)/2
  - ∵ H 的横坐标与 O′ 相同，为 3t/2，且 E 在 H 左侧
  - ∴ E 的横坐标为 3t/2－EH＝3t/2－(t－4)/2＝t＋2
  - ∴ CE＝(t＋2)－6＝t－4
  - ∴ S△CEO′＝√3(t－4)²/4
  - ∴ S＝S△MFO′－S△MDA′－S△CEO′＝√3[15－(t－7)²]/4
  - ∵ 6＜t＜8
  - ∴ 当 t＝7 时，S 取得最大值 15√3/4
- 当前结论：五边形阶段的最大面积为 15√3/4。

### Step 4
- 标题：由右侧边界求最小值
- 目标：求右端边界对应的面积下界。
- 推导：
  - ∵ t＝8 时右侧四边形刚开始，S＝7√3/2
  - ∴ 右侧继续向右移动时，面积继续减小
  - 取右端边界 t＝10 作候选
  - ∵ 此时 A′ 与 B 重合，AA′＝AB＝4√3
  - ∵ 折痕 MN 是 AA′ 的垂直平分线，D 在 AA′ 上
  - ∴ A′D＝AA′/2＝2√3
  - ∵ A′ 与 B 重合，∠CA′A＝30°，且 MN⊥AA′
  - ∴ 右端小三角形是直角三角形，且 A′ 处锐角为 30°
  - ∴ 另一条直角边＝A′D·tan30°＝2
  - ∴ S＝1/2×A′D×2＝1/2×2√3×2＝2√3
  - ∵ t＝10 不在 4＜t＜10 内
  - ∴ 右端下界为 2√3，取不到
- 当前结论：右端下界为 2√3，且取不到。

### Step 5
- 标题：合并答案
- 目标：合并下界与最大值，写出 S 的取值范围。
- 推导：
  - ∵ 左端下界为 2√3，右端下界也为 2√3，且都取不到
  - ∴ S＞2√3
  - ∵ 五边形阶段最大值为 15√3/4，且 t＝7 时可以取到
  - ∴ S≤15√3/4
  - ∴ 2√3＜S≤15√3/4
- 当前结论：2√3＜S≤15√3/4。
