# 图形快照脚本

## 整题层

- 常驻：正方形 ABCD（A 左上、D 右上、B 左下、C 右下；边长设为 1）。
- 静态规则：**所有步骤均为静态图，没有滑块、没有动效**。第（1）问、第（2）问、第（3）问各使用一张固定的几何配置图，不同步骤之间通过高亮、颜色、附加标记区分。
- 图层规则：
  - 第（1）问所有步骤共用一套配置：图 3（中线 EF + 折痕 BG + 像 H）。
  - 第（2）问所有步骤共用一套配置：图 7 典型情形（对角线 AC + 折痕 BE、BG + 像 F 在 BD 上 + 共线点 E、F、G + 交点 H、Q），取 m = √2 − 1（对称配置，便于显示对称结论）。
  - 第（3）问所有步骤共用一套配置：与第（2）问同结构，但取 m = 1/2（n = 1/3）作为代表图，避免与第（2）的对称配置混淆，并提示"图为代表情形，公式适用所有 0 < m < 1"。
- 标注规则：当前推导用到的角、长度、相似三角形高亮（颜色填充 / 边加粗 / 角度标记）；正在求的答案不要直接标在主图上，放右上角结论框里。

## 第（1）问

### 子题层（第（1）问通用）

- 在正方形 ABCD 上叠加：
  - 中线 EF（水平虚线）：E 是 AB 中点（AB 是左边）、F 是 DC 中点（DC 是右边）。
  - 折痕 BG（实线）：B 到 G，G 在 AD 上。
  - 折叠像 H：在 EF 上的某一点（A 沿 BG 翻折后的位置）。
  - 折叠前 △ABG（浅色填充，原状态）和折叠后 △HBG（用浅蓝色填充覆盖）。

### Step 1（q1s1）

- 对应解题步骤：第（1）问 Step 1（由对折的中垂线性质得 AH = BH）
- 当前高亮：
  - EF（中线，加粗虚线）
  - AB（被中线垂直平分的边）
  - 标记 AE = BE 的等长记号
  - 在 E 处加直角符号（EF ⊥ AB）
  - **AH 与 BH 用红色加粗**，强调"H 在 EF 上 ⇒ AH = BH"
- 新增辅助元素：直角符号在 E 处；等长记号在 AE、BE 上。
- 图上标注：（保持简洁，不写"AH = BH"等结论文字，结论放右上角框）
- 右上角结论框：
  - EF 是 AB 的中垂线
  - AH = BH

### Step 2（q1s2）

- 对应解题步骤：第（1）问 Step 2（由折叠 △ABG ≅ △HBG 得 BA = BH）
- 当前高亮：
  - △ABG（浅蓝色填充，标"原"）
  - △HBG（浅黄色填充，标"折后"）
  - BG（折痕，加粗实线）
  - 在 BA 与 BH 上加等长记号
- 新增辅助元素：BG 的折痕标记；BA = BH 的等长符号。
- 退场：上一步 EF 处的直角标记淡化（保留虚线但不加粗）。
- 右上角结论框：
  - △ABG ≅ △HBG
  - BA = BH
  - ∠ABG = ∠HBG

### Step 3（q1s3）

- 对应解题步骤：第（1）问 Step 3（由三段相等证 △ABH 等边，求 ∠ABG）
- 当前高亮：
  - △ABH（三边 AB、BH、AH 各加等长记号，强调三段相等）
  - ∠ABH 处标 60°（等边三角形内角）
  - ∠ABG 处标 30°（折叠对称所得）
- 退场：折痕 BG 与 △HBG 填充保留但减淡；△ABG 填充移除（避免颜色干扰）。
- 右上角结论框：
  - AH = BH = BA
  - △ABH 是等边三角形
  - ∠ABH = 60°
  - ∠ABG = 30°（最终答案）

## 第（2）问

### 子题层（第（2）问通用）

- 在正方形 ABCD 上叠加（图 7 典型配置，m = √2 − 1）：
  - 对角线 AC（虚线，标"折痕 AC"）。
  - 对角线 BD（淡虚线，仅作 F 位置参考）。
  - E 在 AD 上；G 在 CD 上（对称位置 CG = AE = √2 − 1）。
  - F 在 BD 上、距 B 为 1。
  - 折痕 BE（实线，B 到 E）；折痕 BG（实线，B 到 G）。
  - 共线点 E、F、G（连成虚线 EFG，强调三点共线）。
  - H = BE ∩ AC（在 AC 上、靠 A 一侧）；Q = BG ∩ AC（在 AC 上、靠 C 一侧）。

### Step 1（q2s1）

- 对应解题步骤：第（2）问 Step 1（由对折得 ∠ACB = ∠ACD = 45°）
- 当前高亮：
  - 对角线 AC（加粗虚线）。
  - 在 C 处用两个角标分别标 ∠ACB = 45° 和 ∠ACD = 45°。
- 暂时隐藏：BE、BG、F、E、G、H、Q（本步只需要 AC 与正方形）。
- 右上角结论框：
  - ∠ACB = ∠ACD = 45°

### Step 2（q2s2）

- 对应解题步骤：第（2）问 Step 2（列出两次折叠的全等与等角关系）
- 当前高亮：
  - △ABE（浅蓝填充）与 △FBE（浅蓝边线、淡填充）— 折叠前/后对应。
  - △CBG（浅黄填充）与 △FBG（浅黄边线、淡填充）— 折叠前/后对应。
  - 在 A、F、C 三处分别标直角符号（∠BAE = ∠BFE = ∠BFG = ∠BCG = 90°）。
  - 在 B 处用相同弧标 ∠ABE = ∠FBE，用另一相同弧标 ∠CBG = ∠FBG。
- 新增辅助元素：F 点显示出来；BE、BG、EFG 共线虚线全部画出。
- 右上角结论框：
  - △ABE ≅ △FBE
  - △CBG ≅ △FBG
  - 在 F 处：∠BFE = ∠BFG = 90°
  - BF = BA = BC

### Step 3（q2s3）

- 对应解题步骤：第（2）问 Step 3（证 E、F、G 三点共线）
- 当前高亮：
  - F 处的两个直角 ∠BFE 和 ∠BFG（直角符号加粗）。
  - 直线 EFG（红色加粗实线，覆盖原虚线）。
  - 在 F 处标 ∠EFG = 180°（平角标记）。
- 退场：△ABE / △CBG 等填充淡化；BE/BG 折痕保留但减淡。
- 右上角结论框：
  - ∠BFE + ∠BFG = 180°
  - E、F、G 三点共线
  - F 在线段 EG 上

### Step 4（q2s4）

- 对应解题步骤：第（2）问 Step 4（证 ∠EBG = 45°）
- 当前高亮：
  - 在 B 处用四种不同填色标 4 个小角：∠ABE = α、∠EBF = α、∠FBG = β、∠GBC = β（同色表示相等）。
  - ∠ABC = 90° 的整体直角符号包在 4 个小角外围。
  - ∠EBG 用粗弧线标，写"= 45°"。
- 退场：F 处的直角符号、EFG 实线减淡。
- 右上角结论框：
  - α + α + β + β = 90°
  - α + β = 45°
  - ∠EBG = 45°

### Step 5（q2s5）

- 对应解题步骤：第（2）问 Step 5（由 AA 得 △BQH ∽ △CQG）
- 当前高亮：
  - △BQH（浅蓝填充）与 △CQG（浅黄填充）— "X 同侧"对应相似。
  - 在 Q 处标对顶角 ∠BQH = ∠CQG（用同色弧线）。
  - 在 B 处标 ∠QBH = 45°；在 C 处标 ∠QCG = 45°。
- 退场：F、EFG 共线虚线、4 角拆分等可减淡，让出版面。
- 右上角结论框：
  - △BQH ∽ △CQG（AA）
  - BQ/CQ = HQ/GQ

### Step 6（q2s6）

- 对应解题步骤：第（2）问 Step 6（证 △BQC ∽ △HQG，∠BHG = 90°）
- 当前高亮：
  - △BQC（浅紫填充）与 △HQG（浅绿填充）— "X 跨侧"对应相似。
  - Q 处对顶角 ∠BQC = ∠HQG。
  - 在 G 处标 ∠HGQ = ∠HGB = 45°。
  - **在 H 处标 ∠BHG = 90°（红色直角符号，重点）**。
- 退场：上一步的 △BQH/△CQG 填充淡化但保留轮廓。
- 右上角结论框：
  - △BQC ∽ △HQG（SAS）
  - ∠HGB = 45°
  - ∠BHG = 90°

### Step 7（q2s7）

- 对应解题步骤：第（2）问 Step 7（由 Rt△BHG 与对称的 Rt△BQE 得 BH/BG = BQ/BE = √2/2）
- 当前高亮：
  - **Rt△BHG**（浅蓝填充，∠BHG = 90° 直角符号在 H 处）。
  - **Rt△BQE**（浅黄填充，∠BQE = 90° 直角符号在 Q 处）。
  - 在 B 处标 ∠HBG = ∠QBE = 45°（公共角）。
  - 在两个直角三角形上标 BH/BG = √2/2 和 BQ/BE = √2/2 的比例（用小卡片附在三角形旁，不直接写在图上）。
- 退场：上一步的 △BQC/△HQG 填充移除。
- 右上角结论框：
  - BH/BG = √2/2
  - BQ/BE = √2/2

### Step 8（q2s8）

- 对应解题步骤：第（2）问 Step 8（由 SAS 得 △BQH ∽ △BEG，求 HQ/GE）
- 当前高亮：
  - △BQH（浅蓝填充）与 △BEG（浅黄填充）— 一对相似。
  - 公共角 ∠EBG（在 B 处用粗弧标）。
  - HQ 与 GE 用红色加粗，强调最终的比值。
- 退场：直角符号、其他相似三角形填充全部淡化。
- 右上角结论框：
  - △BQH ∽ △BEG（SAS）
  - HQ/GE = BH/BG = √2/2（**最终答案**）

## 第（3）问

### 子题层（第（3）问通用）

- 在正方形 ABCD 上叠加：与第（2）问同样的图 7 配置，**但取代表情形 m = 1/2、n = 1/3 来画图**（这样 F 不在 BD 上，与第（2）的对称配置形成对比，强调"第（3）问是一般 m"）。
- 显式标记：AE = m，CG = n，并在子题层右上角加注"图为 m = 1/2 的代表，公式答案适用所有 0 < m < 1"。
- 已得结论层（来自第（2）问 Step 8）：HQ/GE = √2/2 始终成立。

### Step 1（q3s1）

- 对应解题步骤：第（3）问 Step 1（设 CG = n，EG = m + n）
- 当前高亮：
  - AE = m（在 AD 上 E 处加标记）。
  - CG = n（在 CD 上 G 处加标记）。
  - DG = 1 − n（在 DG 上加标记）。
  - EG = m + n（在 EG 线段上加标记，强调 = AE + CG，并附小注"折叠保距"）。
- 新增辅助元素：在 E 处和 G 处的尺寸线 + 数值标。
- 右上角结论框：
  - DG = 1 − n
  - EG = m + n（= AE + CG）

### Step 2（q3s2）

- 对应解题步骤：第（3）问 Step 2（在 Rt△DEG 中由勾股定理求 n）
- 当前高亮：
  - **Rt△DEG**（浅蓝填充，在 D 处标直角符号）。
  - 三边长度 DE = 1 − m、DG = 1 − n、EG = m + n 全部标在三角形旁。
  - 推导卡片附在主图右下：
    - (1 − m)² + (1 − n)² = (m + n)²
    - 展开：2 − 2m − 2n = 2mn
    - n = (1 − m) / (1 + m)
- 右上角结论框：
  - n = (1 − m) / (1 + m)

### Step 3（q3s3）

- 对应解题步骤：第（3）问 Step 3（由（2）问求 HQ）
- 当前高亮：
  - **HQ 用红色加粗**（在 AC 上 H 到 Q 的一段）。
  - EG 用浅色保留，强调 HQ = (√2/2) · EG。
  - 推导卡片：
    - HQ = (√2/2) · (m + n)
    - m + n = (1 + m²) / (1 + m)
    - HQ = (√2/2) · (1 + m²) / (1 + m)
- 退场：Rt△DEG 填充淡化，只留轮廓。
- 右上角结论框：
  - HQ = (√2/2) · (1 + m²) / (1 + m)

### Step 4（q3s4）

- 对应解题步骤：第（3）问 Step 4（由 AB ∥ CD 求 QC）
- 当前高亮：
  - AB（左边）与 CD（右边）用平行线箭头标记。
  - **QC 用红色加粗**（在 AC 上 Q 到 C 的一段）。
  - CG 与 QC 之间标比例关系 CG/AB = QC/QA（用文字注释或小公式卡）。
  - 推导卡片：
    - n / (1 + n) = QC / √2 （由合比定理）
    - n / (1 + n) = (1 − m) / 2
    - QC = (√2/2) · (1 − m)
- 退场：Step 3 的 HQ 高亮保留但颜色减淡，让 QC 突出。
- 右上角结论框：
  - QC = (√2/2) · (1 − m)

### Step 5（q3s5）

- 对应解题步骤：第（3）问 Step 5（求 HQ/QC）
- 当前高亮：
  - **HQ 与 QC 同时用红色加粗**（一上一下，比值清晰）。
  - 在主图右下用大字写最终公式：
    - HQ/QC = (1 + m²) / (1 − m²)
- 右上角结论框：
  - HQ = (√2/2)(1 + m²) / (1 + m)
  - QC = (√2/2)(1 − m)
  - **HQ/QC = (1 + m²) / (1 − m²)（最终答案）**
